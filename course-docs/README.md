# Course Documentation

This directory contains official course documentation used to define project expectations, assessment criteria, and evaluation standards for BIFX-546: Machine Learning for Bioinformatics.

These documents support the Design and Evaluation phases of the course and are used by the instructor to assess learner progress, provide feedback, and ensure consistency and fairness across project milestones.

## Contents

- **BIFX_546_ML_For_Bioinformatics.pdf**  
  Course syllabus defining scope, learning objectives, weekly structure, policies, and overall course expectations.

- **BIFX_546_Proj_Deliverables_and_Grading_Req.pdf**  
  Defines required project milestones, deliverables, timelines, and submission expectations.

- **BIFX_546_Project_Rubric.pdf**  
  Describes detailed evaluation criteria and performance levels used to assess student work at each milestone.

## Intended Audience

- Instructors and teaching staff  
- Reviewers evaluating course structure and assessment design  

Students should refer to these documents when planning and executing their course projects.
